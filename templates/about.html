üéØ Project Overview

This web application uses deep learning to automatically detect and classify marine plastic waste in ocean environments. The system can identify three categories of waste: Plastic, Organic, and Other materials.

ü§ñ Technology Stack

  • Deep Learning Framework: TensorFlow/Keras
  • Model Architecture: MobileNetV2 (Transfer Learning)
  • Web Framework: Flask
  • Image Processing: Pillow, NumPy
  • Backend: Python

üî¨ Model Details

The classifier uses a Convolutional Neural Network (CNN) based on the MobileNetV2 architecture, fine-tuned for marine waste classification. The model was trained with:

  • Data augmentation for improved generalization
  • Transfer learning from ImageNet pre-trained weights
  • Image normalization and preprocessing
  • 224x224 pixel input resolution

üåç Environmental Impact

Marine plastic pollution is one of the most pressing environmental challenges of our time. An estimated 8 million tons of plastic enter the oceans every year, harming marine life and ecosystems. This tool aims to support:

  • Automated waste monitoring and classification
  • Research on marine pollution patterns
  • Environmental cleanup operations
  • Educational initiatives on ocean conservation
